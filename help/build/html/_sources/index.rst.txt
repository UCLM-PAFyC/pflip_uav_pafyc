Photogrammetric Flight Planning for Unmanned Aerial Vehicle (PFliP UAV) documentation
=====================================================================================

Contenidos:

* `Gestión de proyectos PFLiP UAV <https://www.loom.com/share/97208017354541af89dc1b66ef0a050f>`_
   * Nuevo proyecto
   * Abrir proyecto
   * Edición de proyecto
